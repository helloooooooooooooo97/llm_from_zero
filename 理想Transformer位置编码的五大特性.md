# 理想Transformer位置编码的五大特性

| 特性名称             | 具体描述          |
|----------------------|-----------------------|
| 表达相对位置关系     | 能直接反映Token间的相对距离，$Attention(Q_i,K_j)$ 能体现 $i-j$，相近词联系大，远距离联系弱。                                   |
| 平移不变性或周期性   | 能捕捉序列中的重复或周期性模式（如语法/短语结构），句子前面插入内容时，后续词的相关性不变。                                          |
| 值域有界、尺度合理   | 编码的数值范围有界限，与词嵌入空间尺度相当，不破坏语义空间。                |
| 可泛化到任意长度     | 能支持任意长度的序列，编码方式不局限于训练时的最大长度，如正弦编码通过公式可扩展到任意长度。                                     |
| 唯一性               | 每个位置的编码均足够不同，防止位置混淆。     |
